04 — FAQ

常见问题

企业关心的资质办理高频问题解答

Q
建筑资质代办一般需要多少钱？
▾
资质代办费用没有固定标准，主要看资质等级、专业类别和人员配置情况。总承包资质及升级业务因人员配置多、周期长，费用相对较高；专包资质和劳务资质代办相对经济。费用包含人员配置、资料整理、申报报审等全套服务。建议联系专业顾问，根据企业实际情况一站式核算透明报价。
Q
资质代办需要多长时间？
▾
办理周期因资质类型和等级不同差异明显：专包资质新办约2-3个月，总承包三级资质约3-5个月，资质升级视业绩情况约6-12个月。周期主要耗费在人员招聘配置、社保缴纳（需满3个月）和审批排期等环节。
Q
代办靠谱吗，有哪些注意事项？
▾
正规资质代办是靠谱的，关键在于选对机构。注意事项：①务必签订正式服务合同，明确费用、周期和退款条款；②拒绝口头承诺，拒绝低价套路；③核实机构资质和口碑；④所有申报材料必须真实合规。
Q
企业需要配合提供哪些资料？
▾
企业主要配合提供：营业执照、公司章程、法人身份证、办公场地证明、银行开户信息等基础材料。升级增项还需提供过往工程业绩和原有资质证书。其余申报工作全部由代办机构完成。
Q
资质和安全生产许可证要一起办吗？
▾
建筑施工资质与安全生产许可证属于联动关系，取得施工资质后须在规定时限内完成安许办理，否则资质无法实际使用。建议由同一机构一站式办理可以节省时间。
Q
资质代办失败可以退款吗？
▾
退款取决于合同约定和失败原因：①因代办机构材料编制错误导致申报失败，一般可协商退费或免费重新申报；②因企业提供虚假资料被查出，代办机构不承担责任；③因政策调整等不可抗力，视合同条款协商。
Q
资质办下来后还需要维护吗？
▾
资质到手后需持续维护：①资质证书有效期5年，到期前需申请延期；②建造师等人员离职需及时补充替换；③社保需持续缴纳；④安许有效期3年，到期前3个月申请延期；⑤年报、月报需按时提交。
Q
建筑劳务资质现在怎么办理？
▾
劳务资质已改为备案制，流程大幅简化：登录住建信息化平台注册企业账号，填写劳务资质备案信息，上传营业执照、技术负责人证明、技术工人证书等材料，系统自动审核。
Q
建筑工程施工总承包资质有几个等级？
▾
建筑工程施工总承包资质分为特级、一级、二级、三级，共四个等级。新企业一般从三级开始申请，符合业绩和人员条件后可以逐步升级。
Q
办理建筑资质需要什么人员？
▾
主要需要建造师、技术负责人、技术工人、职称人员、九大员等，根据不同资质类别要求不同，人员数量和专业也有所区别。
Q
建筑资质升级需要什么条件？
▾
资质升级主要考察：企业工程业绩、企业资产、企业主要人员（建造师、职称人员等）。不同资质类别和等级要求不同，具体以最新标准为准。
Q
建筑资质可以跨省使用吗？
▾
建筑资质在全国范围内通用，但在某些地区可能需要按当地要求办理入省/入市备案手续，具体以当地住建部门规定为准。
Q
安全生产许可证需要哪些人员？
▾
主要需要A证（企业主要负责人）、B证（项目负责人/建造师）、C证（专职安全员），人员数量根据资质等级不同而不同。
Q
办理资质需要人员缴纳社保吗？
▾
是的，申报资质时主要人员需要在本企业缴纳社保，一般要求社保连续缴纳3个月以上，部分地区有不同要求。
Q
资质过期了还能延期吗？
▾
资质过期后一般不能延期，需要重新办理。建议在有效期满60日前提前申请延期，避免资质失效影响业务。
Q
公司变更名称后资质怎么办？
▾
公司名称变更后，需要及时办理资质证书变更手续，提供工商变更证明等材料，到住建部门申请变更资质证书信息。
Q
没有资质可以承接工程吗？
▾
不可以，国家规定施工企业必须在资质许可范围内承接工程，无资质承接工程属于违法行为，面临罚款、停业整顿等处罚。
Q
资质可以转让买卖吗？
▾
资质本身不能单独转让，但可以通过企业合并、分立、股权变更等方式实现资质平移，需按规定办理变更手续。
Q
市政公用工程总承包资质能做什么？
▾
可承接城市道路、桥梁、隧道、公共广场、供水、排水、燃气、热力、园林绿化等市政工程施工，根据资质等级不同承接规模不同。
Q
装修装饰资质需要哪些人员？
▾
主要需要建筑工程专业注册建造师、建筑美术设计、结构、暖通、给排水、电气等专业职称人员，以及技术工人等。
Q
钢结构工程专业承包资质几级可以办？
▾
新企业可以直接申请三级资质，已有三级资质且符合业绩条件可申请二级，以此类推，符合条件也可直接申请二级（部分地区开放）。
Q
电子与智能化资质能做什么项目？
▾
可承接电子工程、建筑智能化工程，包括计算机系统、通信、监控、门禁、停车场管理、消防报警等系统工程。
Q
消防设施工程专业承包资质难办吗？
▾
只要满足人员、资产等条件，按标准准备材料即可申请。找专业代办机构可以提高通过率，加快办理速度。
Q
防水防腐保温资质二级需要什么条件？
▾
需要企业净资产400万元以上，技术负责人具有5年以上工程管理经验，建造师3人，技术工人20人等，并满足工程业绩要求。
Q
建筑幕墙工程专业承包资质人员要求？
▾
需要建筑工程、机电工程专业注册建造师，结构、机械等专业职称人员，以及技术工人等，根据等级要求不同而不同。
Q
模板脚手架资质是专业承包吗？
▾
是的，模板脚手架属于专业承包资质，且不分等级，可承担各类模板、脚手架工程的搭设和拆除。
Q
地基基础工程专业承包能做桩基础吗？
▾
可以，地基基础资质可承担各类地基、基础工程施工，包括桩基础、地基处理、基坑支护、降水等工程。
Q
起重设备安装工程专业承包资质需要技术工人吗？
▾
需要，需要起重信号工、起重安装拆卸工等技术工人，三级资质要求10人，二级要求15人，一级要求20人。
Q
预拌混凝土专业承包资质有等级吗？
▾
预拌混凝土专业承包资质目前不分等级，可生产各种强度等级的混凝土和特种混凝土。
Q
桥梁工程专业承包资质能承包哪些工程？
▾
可承担各类桥梁工程施工，包括公路桥梁、市政桥梁、铁路桥梁等，根据资质等级不同，承接工程的单跨跨度和总长有所限制。
Q
隧道工程专业承包资质需要建造师吗？
▾
需要，三级资质要求3名公路工程或铁路工程或市政公用工程或矿业工程专业建造师，二级要求5名，一级要求8名。
Q
公路路面工程专业承包资质几级可以办？
▾
新企业可以申请三级资质，三级资质可承担二级以下公路路面工程施工，二级可承担一级以下路面工程，一级可承担各类路面工程。
Q
公路路基工程专业承包资质与总承包区别？
▾
总承包资质可以承接整个公路项目，路基专业承包只能承接公路路基工程部分，总承包资质覆盖范围更广。
Q
公路交通工程专业承包资质分几个分项？
▾
公路交通工程专业承包资质分为公路安全设施分项和公路机电工程分项，各分为一级、二级。
Q
铁路工程施工总承包资质需要铁路建造师吗？
▾
是的，必须是铁路工程专业注册建造师，三级资质要求3人，二级要求8人，一级要求10人。
Q
港口与航道工程总承包资质申请条件？
▾
需要港口与航道工程专业建造师、相关专业职称人员，以及相应的工程业绩，具体以最新标准为准。
Q
水利水电工程施工总承包资质能做哪些？
▾
可承担水库、水电站、泵站、堤防、河道整治、引调水、灌溉排水、水力发电等水利水电工程施工。
Q
电力工程施工总承包资质需要什么人员？
▾
主要需要机电工程专业注册建造师、电力工程相关专业职称人员、技术工人等，不同等级要求不同。
Q
矿山工程施工总承包资质好办吗？
▾
矿山资质要求相对较高，需要矿业工程建造师、相关专业职称人员，且对技术装备有要求，建议找专业代办机构协助。
Q
冶金工程施工总承包资质包含哪些？
▾
冶金工程总承包资质可承接钢铁冶炼、有色冶炼、冶金机电设备安装等工程，根据等级不同承接规模不同。
Q
石油化工工程施工总承包资质需要化工建造师吗？
▾
需要，要求机电工程（或石油化工）专业注册建造师，三级资质要求5人，二级要求10人，一级要求12人。
Q
通信工程施工总承包资质能做电信工程吗？
▾
可以，通信工程总承包资质可承接各类通信、信息网络工程施工，包括通信线路、通信设备安装、信息化等。
Q
机电工程施工总承包资质包含哪些专业？
▾
机电工程总承包涵盖机械设备安装、电气、管道、通风空调等工程，可承接各类机电工程施工。
Q
古建筑工程专业承包资质需要什么人员？
▾
需要建筑工程专业建造师、建筑、结构等专业职称人员，以及传统技艺的石工、瓦工、木工、彩绘工等技术工人。
Q
城市及道路照明工程专业承包资质能做什么？
▾
可承接城市道路、桥梁、广场、公园、公共绿地等各类城市道路照明工程施工。
Q
环保工程专业承包资质办理条件？
▾
需要注册建造师、环保、结构等职称人员、技术工人，满足净资产要求，三级还可新申请。
Q
特种工程专业承包资质包含哪些？
▾
特种工程专业承包资质不分等级，包括建筑物纠偏和平移、结构补强、特殊设备起重吊装、特种防雷技术等。
Q
工程监理资质有几个专业类别？
▾
工程监理资质有房屋建筑工程、市政公用工程、电力工程、水利水电工程、公路工程等多个专业类别，各分甲级、乙级。
Q
工程设计资质有哪些等级？
▾
工程设计资质有综合资质（不分级）、行业资质（甲、乙级）、专业资质（甲、乙、丙级）、专项资质等。
Q
工程造价咨询资质可以做什么？
▾
可承担建设项目建议书与可行性研究投资估算、工程概算、预算、结算、竣工决算等造价咨询业务。
Q
安全生产许可证有效期几年？
▾
安全生产许可证有效期为3年，有效期满需要延期的，应当在期满前3个月向原审批机关申请延期手续。
Q
安许过期多久会被注销？
▾
安全生产许可证过期后一般自动失效，建议提前3个月申请延期，避免失效影响工程承接。
Q
安许延期需要重新审查条件吗？
▾
安许有效期内严格遵守安全生产法规、无死亡事故等，经原审批机关同意可直接延期，否则需要重新审查。
Q
安全员A/B/C证有什么区别？
▾
A证是企业主要负责人（法人、总经理、技术负责人等）；B证是项目负责人（建造师）；C证是专职安全员。
Q
建筑施工企业资质申请流程是怎样的？
▾
大致流程是：企业注册→人员招聘配置→缴纳社保→准备材料→申报→初审→公示→领证，建议找专业机构代办。
Q
资质申报需要哪些纸质材料？
▾
现在很多地方实行电子化申报，具体以当地要求为准，一般需营业执照、人员证书、社保缴纳证明、业绩材料等。
Q
建筑资质在哪里申请办理？
▾
一般在企业注册地的住建部门（住房和城乡建设局）申请，部分高等级资质需到省住建厅或住建部审批。
Q
资质公示后多久能拿到证书？
▾
公示无异议后一般1-2周可以领证，具体时间以当地住建部门工作效率为准，有些地方支持电子证书。
Q
资质证书遗失了可以补办吗？
▾
可以，资质证书遗失后应及时向原发证机关申请补办，提交遗失声明等材料，现在一般可直接在线申请补发电子证书。
Q
一个公司可以办理多个资质吗？
▾
可以，企业可以根据业务需要同时办理多个资质，人员可以在不同资质间共用，但需满足各资质的要求。
Q
技术工人需要持证上岗吗？
▾
是的，技术工人需要持有相应的职业技能等级证书或岗位培训合格证书，根据资质要求配备相应数量。
Q
职称人员有专业要求吗？
▾
有，不同资质类别要求不同专业的职称人员，如建筑工程资质要求结构、给排水、暖通、电气等专业。
Q
建造师只能用于一个资质吗？
▾
不是，建造师可以用于多个资质，只要满足各资质的专业和数量要求，一名建造师可同时作为多个资质的人员。
Q
技术负责人需要什么条件？
▾
技术负责人需要具有相关专业高级职称或注册建造师执业资格，且具有一定年限的工程管理经验，具体看资质标准。
Q
企业净资产如何考核？
▾
以企业申请资质前一年度或当期经审计的财务报表中净资产指标为准，新企业以营业执照载明的注册资本为准。
Q
工程业绩如何认定？
▾
一般需要提供中标通知书、合同、竣工验收文件等材料，部分地区需要在四库一平台或当地平台备案。
Q
工程业绩需要备案吗？
▾
建议及时备案，四库一平台业绩备案对资质升级很重要，很多地方升级时要求平台可查业绩。
Q
四库一平台是什么？
▾
四库一平台是全国建筑市场监管公共服务平台，包括企业库、人员库、项目库、信用库，统一平台。
Q
什么是建筑企业资质告知承诺制？
▾
告知承诺制是指企业承诺符合条件，主管部门直接审批，事后核查，简化流程但加强事后监管。
Q
资质告知承诺制核查不合格怎么办？
▾
如果事后核查发现不符合条件，可能会撤销资质，因此承诺要符合实际情况，建议申报前做好准备。
Q
建筑资质挂靠合法吗？
▾
资质挂靠属于违法行为，国家严厉打击挂靠行为，建议企业通过合法渠道办理资质，避免法律风险。
Q
什么是转包和违法分包？
▾
转包是承包单位将工程转给他人，违法分包是不符合规定的分包，都属于违法行为，面临处罚。
Q
建筑企业信用分重要吗？
▾
非常重要，信用分影响投标、资质动态核查、资质升级等，信用差的企业可能面临扣分甚至资质撤销。
Q
什么是资质动态核查？
▾
动态核查是住建部门定期或不定期核查企业是否仍然符合资质条件，包括人员、资产等，不合格会要求整改。
Q
资质动态核查不合格会怎样？
▾
不合格会被要求限期整改，整改不合格的可能降低资质等级或吊销资质证书，因此平时要做好资质维护。
Q
建筑企业年报是什么？
▾
建筑企业需要每年按要求上报年报，包括企业基本信息、人员、项目等情况，不按时上报可能影响资质。
Q
资质可以借给别的公司用吗？
▾
不可以，资质外借、挂靠属于违法违规行为，会面临处罚，严重的会吊销资质证书。
Q
资质分立和合并有什么区别？
▾
分立是一个企业分成多个，合并是多个企业合成一个，都可以按规定进行资质重组。
Q
如何选择靠谱的资质代办公司？
▾
建议看成立时间、成功案例、口碑、是否有正式办公地址、合同是否规范、是否有专业团队等。
Q
资质代办一般有哪些服务内容？
▾
一般包括人才推荐或配置、材料准备、申报、沟通协调、后续维护指导等全程或部分服务。
Q
资质代办需要法人到场吗？
▾
一般不需要法人到场，有些地方在领取资质或个别环节可能需要，大部分工作代办机构可以完成。
Q
代办资质不成功可以退款吗？
▾
要看合同约定，建议签订合同时明确退款条款，正规代办机构一般会有退款保障条款。
Q
资质增项和升级可以同时办吗？
▾
可以，只要满足相应条件，可以同时申请资质增项和资质升级，建议咨询专业顾问确认条件。
Q
资质增项需要人员重新配置吗？
▾
如果现有人员已经满足增项资质要求，不需要重新配置，不满足的话需要补充相应人员。
Q
新成立的公司可以直接申请二级资质吗？
▾
大部分地区还是从三级开始申请，但部分地区对部分资质开放直接申请二级，具体以当地最新政策为准。
Q
建筑企业如何做好资质维护？
▾
做好人员管理、按时社保、及时业绩备案、按时年报、关注政策变化、有效期满前及时延期。
Q
资质年检取消了吗？
▾
是的，传统的资质年检已取消，改为动态核查和年度报告制度，企业要做好日常维护。
Q
哪些情况会导致资质被撤销？
▾
主要有申报材料弄虚作假、不符合资质条件整改不合格、违法违规情节严重、资质有效期满未延期等情况。
Q
什么是建筑企业资质换证？
▾
换证是根据新版资质标准，企业按新条件申请换发新证书，一般在标准更新时集中换证。
Q
电子资质证书和纸质证书效力一样吗？
▾
是的，电子资质证书与纸质证书具有同等法律效力，现在很多地区全面实行电子证书。
Q
如何查询企业资质信息？
▾
可以通过全国建筑市场监管公共服务平台（四库一平台）查询企业资质、人员、业绩等信息。
Q
资质证书变更需要什么材料？
▾
一般需要变更申请表、营业执照、股东会决议、原资质证书等，具体看变更事项。
Q
企业地址变更资质要变更吗？
▾
是的，营业执照地址变更后，资质证书也要及时办理变更手续，保持信息一致。
Q
建筑劳务公司需要办理资质吗？
▾
需要，虽然劳务资质改为备案制，但仍需要办理备案手续，承接工程需要有相应备案。
Q
劳务资质备案需要多久？
▾
劳务资质备案流程简化，材料齐全一般很快可以完成，部分地区当天或几个工作日内完成。
Q
建筑资质是全国通用吗？
▾
是的，建筑资质全国通用，但在外地承接工程可能需要办理入省备案手续。
Q
资质变更法人需要重新考核吗？
▾
不需要重新考核资质条件，只需办理法人信息变更，但新法人需要按要求持有A证。
Q
建筑资质对企业有什么好处？
▾
可以合法承接相应工程、提高竞争力、参与招投标、承接更大项目、建立品牌信任、享受政策优惠。
Q
资质标准多久更新一次？
▾
不固定，住建部会根据行业情况调整资质标准，建议关注最新政策和官方通知。
